# Jab Jab Jab Right Hook

Framework central de Gary Vaynerchuk para equilibrar conteudo de valor e conteudo de venda nas redes sociais.

## Conceito

- **Jab** = Dar valor, entreter, educar, inspirar (sem pedir nada em troca)
- **Right Hook** = Pedir algo (compra, cadastro, clique, compartilhe)

## A Proporcao

Para cada pedido (right hook), voce deve ter dado pelo menos 3 jabs:
- 3 posts de valor para cada 1 post de venda
- Alguns praticam 5:1 ou ate 7:1

## Exemplos de Jabs
- Tutorial gratuito ensinando algo util
- Bastidores do seu negocio
- Dicas praticas que o seguidor pode aplicar hoje
- Responder perguntas da audiencia
- Conteudo inspiracional genuino

## Exemplos de Right Hooks
- "Link na bio para comprar"
- "Se inscreva no curso"
- "Agende uma consulta"
- "Baixe o ebook gratuito" (lead magnet)

## Erros Comuns
1. So fazer right hooks (so vender) - Perde seguidores
2. So fazer jabs (so dar valor) - Nunca converte
3. Right hook sem contexto - Vender algo irrelevante para a audiencia
4. Jab de baixa qualidade - Conteudo generico que nao agrega

## Aplicacao com IA
Use IA para escalar seus jabs:
- Gere variantes de conteudo educacional
- Adapte um jab para 5 plataformas diferentes
- Crie series de conteudo a partir de um tema pilar
- Automatize a programacao de posts (jab-jab-jab-hook)
