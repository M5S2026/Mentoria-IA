# Copy para Lancamentos com IA

## Os 4 Tipos de Copy Essenciais

### 1. Copy de Captura
Para paginas de lead magnet e formularios:
- **Headline:** Promessa clara + especificidade
- **Subheadline:** Detalhe o beneficio principal
- **Bullets:** 3-5 beneficios concretos
- **CTA:** Acao clara ("Quero Receber" > "Enviar")

### 2. Copy de Pre-Lancamento
Para CPLs e emails de aquecimento:
- Conte historias que demonstrem a transformacao
- Use perguntas que gerem reflexao
- Entregue valor real (nao so tease)
- Plante sementes para a oferta sem vender diretamente

### 3. Copy de Vendas
Para pagina de vendas e emails de abertura:
- **Lead:** Gancho emocional nos primeiros 3 segundos
- **Corpo:** Problema > agitacao > solucao
- **Prova:** Depoimentos, numeros, garantia
- **Oferta:** Preco ancorado, bonus, condicoes especiais
- **CTA:** Urgencia real + acao clara

### 4. Copy de Fechamento
Para os ultimos dias do lancamento:
- Reforce escassez (vagas restantes, timer real)
- Resuma os principais beneficios
- Antecipe o arrependimento de nao comprar
- Ultimo email: direto, curto, urgente

## IA como Copiloto de Copy

- Gere 10 variantes de headline e teste
- Crie angulos diferentes para a mesma oferta
- Transforme features em beneficios automaticamente
- Adapte o tom para diferentes segmentos da audiencia
- Revise e melhore copy existente com feedback de IA

## Erros Comuns

1. Copy generica que poderia ser de qualquer nicho
2. Falar de features em vez de beneficios e transformacao
3. Nao usar prova social suficiente
4. CTA fraco ou confuso
5. Deixar a IA escrever copy sem sua voz pessoal
