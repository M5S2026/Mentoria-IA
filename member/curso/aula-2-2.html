Desenvolvimento guiado - construindo o app

Hora de construir o "Minhas Financas"

Com o brief pronto da aula anterior, vamos construir nosso app de financas pessoais passo a passo usando o Claude Code.

Passo 1: Criar a pasta e iniciar

mkdir minhas-financas
cd minhas-financas
claude

Passo 2: Enviar o prompt inicial

Envie este prompt completo para o Claude Code:

Crie um app de financas pessoais chamado "Minhas Financas" em um
unico arquivo HTML.

Funcionalidades:
- Formulario para adicionar transacao: tipo (receita/despesa),
  descricao, categoria (dropdown com opcoes), valor (R$), data
- Lista de transacoes com opcao de deletar
- Resumo: total receitas, total despesas, saldo atual
- Dados salvos no localStorage

Categorias: Alimentacao, Transporte, Moradia, Lazer, Saude,
Educacao, Salario, Freelance, Outros

Design:
- Tema escuro, fundo #0a0a1a
- Cards com efeito glass (rgba branco com backdrop-filter blur)
- Receitas em verde (#00ff88), despesas em vermelho (#e94560)
- Font: system-ui ou sans-serif
- Totalmente responsivo
- Layout: resumo no topo, formulario abaixo, lista de transacoes

O Claude Code vai criar um arquivo index.html completo. Aceite a criacao e abra no navegador para ver o resultado.

Passo 3: Revisar e iterar

Abra o app no navegador. Teste adicionar algumas transacoes. Provavelmente voce vai querer ajustes. Aqui estao exemplos de iteracoes comuns:

O formulario esta bom, mas quero que a data venha preenchida com
a data de hoje por padrao.
Adicione uma animacao suave quando uma nova transacao e adicionada
na lista. Use fadeIn.
O botao de deletar esta muito pequeno no mobile. Aumente o tamanho
e coloque um icone de lixeira em vez de texto.
Ordene as transacoes por data, da mais recente para a mais antiga.
Fluxo de iteracao:

1. Teste no navegador → 2. Anote o que quer mudar → 3. Descreva a mudanca no Claude Code → 4. Aceite e teste novamente. Repita ate ficar satisfeito.

Passo 4: Adicionar filtros

Adicione filtros acima da lista de transacoes:
- Filtrar por tipo: Todos, Receitas, Despesas
- Filtrar por categoria (dropdown)
- Filtrar por mes/ano
Os filtros devem funcionar em combinacao.

Passo 5: Melhorar o resumo

No card de resumo, adicione:
- Icones nos cards (use emojis: receita, despesa, saldo)
- Animacao de contagem nos valores quando a pagina carrega
- Uma barra de progresso mostrando a porcentagem de gastos
  em relacao as receitas

Entendendo o que foi gerado

Observe a estrutura do codigo que o Claude Code criou:

  • No topo: HTML com a estrutura (formulario, divs para resumo e lista)
  • No meio: CSS com todo o estilo visual (cores, layout, responsivo)
  • No final: JavaScript com a logica (adicionar, deletar, salvar, calcular)

Tudo em um unico arquivo HTML - isso facilita o compartilhamento e deploy.

Exercicio

Siga todos os passos acima e construa o app "Minhas Financas". Depois, adicione pelo menos 2 funcionalidades extras que voce achar util (ex: editar transacao, exportar dados, modo claro).
