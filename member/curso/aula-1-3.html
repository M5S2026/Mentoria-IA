Seu primeiro comando

Hora de colocar a mao na massa

Agora que tudo esta instalado, vamos criar algo real. Nesta aula voce vai interagir com o Claude Code pela primeira vez e criar sua primeira pagina web.

Iniciando o Claude Code

Abra o terminal no VS Code (Ctrl+`) e navegue ate sua pasta de projetos:

# Navegue para sua pasta de projetos
cd ~/meus-projetos-claude

# Crie uma pasta para este exercicio
mkdir meu-primeiro-site
cd meu-primeiro-site

# Inicie o Claude Code
claude

Voce vera o prompt do Claude Code esperando sua instrucao. Agora vamos conversar com ele.

Exercicio: Criando uma pagina "Hello World"

Digite o seguinte no Claude Code:

Crie uma pagina HTML simples com o titulo "Meu Primeiro Site" e
um paragrafo dizendo "Ola mundo! Eu criei isso com IA."
Use um fundo azul escuro e texto branco. Centralize tudo na tela.

O Claude Code vai processar seu pedido e mostrar o que ele pretende criar. Voce vera algo como:

Vou criar o arquivo index.html com o conteudo solicitado.

[Claude mostra o codigo que sera criado]

Posso criar este arquivo? (s/n)

O sistema de permissoes

Essa pergunta e muito importante. O Claude Code sempre pede permissao antes de criar ou modificar qualquer arquivo. Isso e uma medida de seguranca - voce tem controle total sobre o que acontece no seu computador.

As opcoes que voce vera:

  • Yes (s) - Aceita a criacao/modificacao
  • No (n) - Rejeita e pode pedir algo diferente
  • Always allow - Aceita automaticamente para este tipo de acao (use com cuidado)

Digite s (ou Yes) para aceitar. O arquivo sera criado!

Visualizando o resultado

Agora vamos ver o que foi criado. Voce tem algumas opcoes:

Opcao 1: Abrir direto no navegador

# Mac
open index.html

# Windows
start index.html

# Linux
xdg-open index.html

Opcao 2: Usar o VS Code Live Server

Instale a extensao "Live Server" no VS Code, clique com botao direito no arquivo index.html e selecione "Open with Live Server". Isso abre o site no navegador e atualiza automaticamente quando voce faz mudancas.

Fazendo modificacoes

O poder real do Claude Code esta nas iteracoes. Ainda na conversa, peca mudancas:

Adicione um botao vermelho abaixo do texto que diz "Clique aqui".
Quando clicar, mude o texto do paragrafo para "Voce clicou! Isso
foi feito com JavaScript."

O Claude Code vai modificar o arquivo existente, adicionando o botao e o JavaScript necessario. Novamente, ele vai pedir permissao antes de salvar.

Continue experimentando:

Agora adicione uma imagem de placeholder no topo da pagina.
Use um servico como placeholder.com com tamanho 300x200.
Adicione um footer com o texto "Feito com Claude Code - 2026"
e centralize.
Dica importante:

Nao tenha medo de experimentar! Se algo der errado, voce pode pedir ao Claude Code para desfazer ("desfaca a ultima mudanca") ou simplesmente descrever o que quer de forma diferente. Cada erro e uma oportunidade de aprender a comunicar melhor.

Entendendo o que foi criado

Mesmo que voce nao precise saber programar, e util entender a estrutura basica:

  • HTML - Define o conteudo (textos, botoes, imagens)
  • CSS - Define a aparencia (cores, tamanhos, posicoes)
  • JavaScript - Define o comportamento (o que acontece ao clicar)

O Claude Code cuida dos tres para voce. Mas saber o nome de cada um ajuda a dar instrucoes mais precisas.

Exercicio final

Antes de ir para a proxima aula, tente criar por conta propria:

  1. Uma pagina com seu nome como titulo
  2. Uma lista de 3 coisas que voce gosta
  3. Um botao que muda a cor do fundo ao clicar

Use o Claude Code para fazer tudo. Descreva o que quer em portugues e veja a magica acontecer.
