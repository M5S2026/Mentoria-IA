Instalacao passo a passo

Pre-requisitos

Antes de instalar o Claude Code, voce precisa de duas coisas no seu computador:

  • Node.js (versao 18 ou superior) - o motor que roda o Claude Code
  • VS Code (ou qualquer editor de texto) - onde voce vai ver os arquivos criados

Passo 1: Instalar o Node.js

O Node.js e o ambiente que permite rodar o Claude Code. Vamos instalar:

No Mac

Abra o Terminal (busque "Terminal" no Spotlight com Cmd+Espaco) e rode:

# Se voce tem Homebrew instalado:
brew install node

# Se nao tem Homebrew, instale primeiro:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node

No Windows

Baixe o instalador em nodejs.org, clique no botao LTS (versao estavel), execute o instalador e siga os passos padrao (Next, Next, Install).

No Linux

# Ubuntu/Debian
sudo apt update
sudo apt install nodejs npm

# Ou use o nvm (recomendado):
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts

Verificar a instalacao

Apos instalar, verifique se deu certo:

node --version
# Deve mostrar algo como: v20.x.x ou v22.x.x

npm --version
# Deve mostrar algo como: 10.x.x
Dica:

Se o comando nao for reconhecido, feche e abra o terminal novamente. As vezes as variaveis de ambiente precisam ser recarregadas.

Passo 2: Instalar o VS Code

Baixe o VS Code em code.visualstudio.com. E gratuito e funciona em Mac, Windows e Linux. Instale normalmente.

Apos instalar, abra o VS Code e familiarize-se com o terminal integrado: use o atalho Ctrl+` (ou Cmd+` no Mac) para abrir o terminal dentro do VS Code. Esse sera seu ambiente de trabalho principal.

Passo 3: Instalar o Claude Code

Agora o passo principal. No terminal, rode:

npm install -g @anthropic-ai/claude-code

Esse comando instala o Claude Code globalmente no seu computador. Aguarde alguns segundos ate finalizar.

Verificar a instalacao

claude --version
# Deve mostrar a versao instalada

Passo 4: Configurar a chave da API

O Claude Code precisa de uma chave de API da Anthropic para funcionar. Siga estes passos:

  1. Acesse console.anthropic.com
  2. Crie uma conta (se ainda nao tem)
  3. Va em "API Keys" no menu lateral
  4. Clique em "Create Key"
  5. Copie a chave gerada (comeca com sk-ant-...)

Agora configure a chave no seu terminal:

# Mac/Linux - adicione ao seu ~/.zshrc ou ~/.bashrc:
export ANTHROPIC_API_KEY="sk-ant-sua-chave-aqui"

# Depois recarregue:
source ~/.zshrc

# Windows (PowerShell):
$env:ANTHROPIC_API_KEY="sk-ant-sua-chave-aqui"
Importante sobre custos:

O Claude Code usa a API da Anthropic que tem custo por uso. Para este curso, o gasto tipico e de US$ 5-15 no total. A Anthropic oferece creditos iniciais gratuitos para novos usuarios. Voce pode definir limites de gasto no painel.

Passo 5: Testar tudo

Vamos verificar se tudo esta funcionando. Crie uma pasta para o curso e inicie o Claude Code:

# Crie uma pasta para seus projetos
mkdir ~/meus-projetos-claude
cd ~/meus-projetos-claude

# Inicie o Claude Code
claude

Se tudo estiver certo, voce vera uma interface de conversa no terminal. Digite "Ola, estou pronto para comecar!" e pressione Enter. Se o Claude responder, parabens - esta tudo configurado!

Para sair, digite /exit ou pressione Ctrl+C.

Resumo da instalacao

  • Node.js instalado e verificado (node --version)
  • VS Code instalado com terminal integrado
  • Claude Code instalado via npm install -g @anthropic-ai/claude-code
  • Chave de API configurada como variavel de ambiente
  • Teste realizado com sucesso
