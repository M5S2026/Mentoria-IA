A arte do prompt para codigo

O segredo esta nas instrucoes

A qualidade do que o Claude Code cria depende diretamente da qualidade das suas instrucoes. Um prompt vago gera um resultado generico. Um prompt especifico gera um resultado profissional.

Prompt ruim vs Prompt bom

Veja a diferenca:

Prompt ruim:

Faca um site bonito.

Problema: "bonito" e subjetivo. O Claude nao sabe suas preferencias de cores, layout ou conteudo.

Prompt bom:

Crie uma landing page para uma consultoria de marketing digital.
- Fundo escuro (#0a0a1a) com texto branco
- Cor de destaque: vermelho (#e94560)
- Secoes: Hero com titulo e CTA, Servicos (3 cards), Depoimentos, Contato
- Design moderno com efeito glass (backdrop-filter blur)
- Responsivo para mobile
- Formulario de contato com nome, email e mensagem

Resultado: O Claude Code cria exatamente o que voce precisa porque tem todas as informacoes.

O Framework CRISP

Use este framework para estruturar seus prompts:

  • C - Contexto: Qual o objetivo? Para quem e? ("Uma ferramenta para donos de restaurantes calcularem custos")
  • R - Requisitos: O que precisa ter? ("Campos para ingredientes, quantidade, preco unitario. Total automatico.")
  • I - Input/Output: O que entra e o que sai? ("Usuario digita dados, sistema calcula custo total e margem de lucro")
  • S - Style: Como deve parecer? ("Design limpo, cores neutras, fonte moderna, responsivo")
  • P - Prioridades: O que e mais importante? ("Foco na usabilidade mobile, calculo preciso e exportar para PDF")

Exemplos praticos por tipo de projeto

Para criar uma landing page:

Crie uma landing page para um curso online de fotografia.

Contexto: Curso de R$ 297, publico-alvo sao iniciantes em fotografia.

Secoes necessarias:
1. Hero: titulo impactante, subtitulo, botao "Quero me inscrever"
2. Problema: 3 cards com dores do publico (equipamento caro, fotos ruins, falta de pratica)
3. Solucao: O que o curso oferece (6 modulos, 30 aulas, certificado)
4. Depoimentos: 3 depoimentos de alunos ficticios
5. Preco: Box com preco, opcao de parcelamento, botao de compra
6. FAQ: 5 perguntas frequentes com accordion
7. Footer: links e redes sociais

Design: fundo escuro, tipografia moderna, imagens placeholder,
totalmente responsivo. Cor principal: dourado (#d4af37).

Para criar uma calculadora:

Crie uma calculadora de ROI para campanhas de marketing.

Campos de entrada:
- Investimento mensal em ads (R$)
- Custo por lead (R$)
- Taxa de conversao (%)
- Ticket medio (R$)

Calculos automaticos:
- Numero de leads gerados
- Numero de vendas
- Receita total
- ROI em porcentagem
- Lucro liquido

Inclua um grafico de barras comparando investimento vs receita.
Botao para exportar resultado em PDF.
Design profissional, fundo escuro, cor de destaque verde.

Para criar um dashboard:

Crie um dashboard de vendas mensais.

Dados: use dados ficticios de 12 meses (janeiro a dezembro).

Graficos:
- Linha: evolucao de vendas mensais
- Pizza: distribuicao por categoria (Eletronicos, Roupas, Alimentos, Outros)
- Barras: comparacao metas vs realizado

Cards no topo: Receita Total, Ticket Medio, Numero de Vendas, Meta Atingida (%).

Use Chart.js para os graficos. Design responsivo, tema escuro.
Regra de ouro:

Quanto mais especifico voce for sobre cores, layout, funcionalidades e dados, melhor sera o resultado. Se voce nao especificar, o Claude Code vai tomar decisoes por voce - e elas podem nao ser o que voce queria.

Erros comuns de iniciantes

  1. Pedir tudo de uma vez: Comece com a estrutura basica, depois va adicionando funcionalidades uma por uma
  2. Nao revisar o resultado: Sempre abra o arquivo no navegador e teste antes de pedir mais mudancas
  3. Descrever de forma ambigua: "Coloque um menu" - onde? Horizontal? Vertical? Com quais links?
  4. Nao dar contexto: "Faca um formulario" - para que? Quais campos? O que acontece ao enviar?
  5. Ter medo de iterar: Nao precisa acertar de primeira. Refine ate ficar perfeito

Tecnicas avancadas de prompt

  • Referencia visual: "Estilo similar ao site da Apple - minimalista, muito espaco em branco, tipografia grande"
  • Especifique tecnologias: "Use Chart.js para graficos", "Use CSS Grid para layout"
  • Peca em etapas: Primeiro a estrutura HTML, depois o estilo, depois a interatividade
  • De exemplos de dados: "Use estes dados: Janeiro R$ 5.000, Fevereiro R$ 7.200..."

Exercicio

Usando o framework CRISP, escreva um prompt para criar uma ferramenta de lista de tarefas (todo list) com as seguintes funcionalidades:

  • Adicionar e remover tarefas
  • Marcar como concluida
  • Filtrar por status (todas, pendentes, concluidas)
  • Salvar no localStorage
  • Design bonito com tema escuro

Escreva o prompt, envie no Claude Code e veja o resultado. Na proxima aula, vamos usar essas tecnicas para planejar um projeto completo.
